Favicon.pub File Package Explanation
===================

This ZIP package contains the following files:

PNG Formats:
- favicon-16x16.png (16x16 pixels, for browser tabs)
- favicon-32x32.png (32x32 pixels, for taskbars)
- favicon-48x48.png (48x48 pixels, for desktop shortcuts)
- android-chrome-192x192.png (192x192 pixels, Android PWA icon)
- android-chrome-512x512.png (512x512 pixels, Android PWA launch icon)

Apple Icons:
- apple-touch-icon.png (180x180 pixels, default iOS icon)

Other Formats:
- favicon.ico (Traditional ICO format with multiple resolutions: 16x16, 32x32, 48x48. Note: Browsers/OS may select the most appropriate size based on display environment, or show smaller sizes due to caching. You can use professional ICO viewers like Axialis IconWorkshop or online ICO inspection tools to verify all sizes are embedded.)
- site.webmanifest (PWA manifest file)

Usage:
Place all files in your website's root directory, then add the appropriate link tags in the <head> section of your HTML.
