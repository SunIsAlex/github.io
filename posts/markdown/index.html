Markdown是一种轻量级标记语言，旨在让用户以易读易写的纯文本格式编写文档，并能够轻松转换为HTML等格式。

一、一级标题#

用# 可以表示一级标题

二、级标题#

用## 可以表示二级标题

三、级标题#

用### 可以表示三级标题

分界线#

用---表示分界线 四、文本格式
用 **粗体** 或 __粗体__ 表示粗体
用 *斜体* 或 _斜体_ 表示斜体
用 ~~删除线~~ 表示删除线

五、列表
无序列表用 *,+ 或 - 开头：

  • 项目一
  • 项目二
    • 子项目（缩进两个空格）

有序列表用数字加英文句点：

  1. 第一项
  2. 第二项
    1. 子项（缩进三个空格）

九、表格
用竖线 | 分隔列，连字符 - 分隔表头与内容：

十、任务列表
用 - [ ] 或 - [x] 表示任务：

  • 未完成
  • 已完成

十一、脚注
用 [^1] 标注脚注位置，在文末用 [^1]: 定义内容1。
